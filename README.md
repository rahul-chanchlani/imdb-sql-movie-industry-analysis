# imdb-sql-movie-industry-analysis
Analysis of global movie industry trends using SQL. Performed complex data extraction and relational database querying to uncover insights on genre performance, regional market dominance, and actor success

# IMDb Movie Industry Analysis

# Project Overview
This project involves a comprehensive exploratory data analysis (EDA) of the global movie industry using a large-scale IMDb database. By executing complex SQL queries, I analyzed trends in production volumes, genre performance, and regional market dominance to extract data-driven insights.

# Business Insights
Market Trends: Identified 2017 as the peak year for global movie production, with a significant surge in market activity.

Regional Dominance: Highlighted the robust market performance of India and the USA, which collectively produced 887 movies in 2019 alone.

Genre Performance: Found that Drama is the most prolific genre, with an average duration of ~107 minutes, suggesting it as the most effective genre for audience engagement.

Production Excellence: Identified top-tier production houses like Star Cinema and Twentieth Century Fox, along with high-impact actors and directors, based on aggregated performance metrics.

# Technical Skills Demonstrated
Complex Data Extraction: Utilized advanced SQL functions to join multiple tables and resolve complex relationships across the database.

Window Functions: Employed ROW_NUMBER(), RANK(), and DENSE_RANK() for competitive performance analysis.

Aggregations & CTEs: Used Common Table Expressions (CTEs) and multi-level aggregations to handle high-volume data processing.

Data Integrity: Performed rigorous data cleaning to identify and document null values in critical financial columns (e.g., worlwide_gross_income).

# Key SQL Techniques Used
Joins: Efficiently integrated data across movie, genre, ratings, and names tables.

Mathematical Operations: Used AVG(), SUM(), and DATEDIFF() to generate comparative performance reports.

Data Segmentation: Applied CASE statements to categorize and clean inconsistent data entries.

# Conclusion
This analysis provides a clear picture of global movie market dynamics, from production bottlenecks to genre popularity. These insights demonstrate an ability to translate raw database records into clear, actionable business intelligence.
